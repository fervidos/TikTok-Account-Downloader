# Contributing

Thanks for considering contributing to TikTok Account Downloader! This project is maintained as a small utility, but contributions are welcome.

## Getting Started

1. Fork the repository and create a feature branch.
2. Install dependencies (see `README.md`).
3. Make small, focused changes with clear commit messages.

## Coding Style

- Keep code readable and simple.
- Prefer explicit behavior over magic.
- Keep dependencies minimal and explained.

## Pull Requests

- Describe what the change does and why.
- Add or update documentation when behavior changes.
- If you introduce a new feature, consider adding a small usage example.

## Issues

If you find a bug or have a feature request, please open an issue with a clear description and reproduction steps.
